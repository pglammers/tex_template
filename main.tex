\boxmain{Purpose of this document}{
    This document provides a LaTeX template, as well as a number of useful tools for
    handling LaTeX projects.
}

Overview of the LaTeX template.

\begin{itemize}
    \item \texttt{preamble/symbols.sty} provides a list of common symbols.
    \item \texttt{preamble/environments.sty} provides common environments.
    \item \texttt{preamble/boxes.sty} provides box environments.
        Not common in published articles, but useful when writing lecture notes.
    \item \texttt{preamble/todos.sty} provides margin comments.
        Useful in general but particularly useful when collaborating.
    \item \texttt{preamble/headings.sty} provides custom section heading styles.
\end{itemize}

Overview of the other files.

\begin{itemize}
    \item \texttt{.vscode/latex.code-snippets} provides snippets for vscode users.
    \item \texttt{pytex\_zip.py} contains Python code to zip the tex document into a single file.
    \item \texttt{pytex\_unzip.py} performs the reverse operation.
    \item \texttt{tex-fmt.toml} provides configuration for running \texttt{tex-fmt ----recursive},
        which is useful for autoformatting the latex code.
    \item \texttt{conventions.md} provides a location to verbally describe writing and
        typesetting conventions within the project.
\end{itemize}
